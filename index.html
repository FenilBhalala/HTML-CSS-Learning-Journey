About This Website:

This website is designed as my digital portfolio — a place to organize, present, and reflect on everything I’ve learned in web development. Instead of spreading my projects across multiple GitHub repositories, I’ve created one central hub to explore them all.

About Me:

I’m Fenil Bhalala, a passionate IT student at Veer Narmad South Gujarat University. I'm currently exploring the world of web development, and this website is part of my practical journey toward becoming a full-stack developer. I enjoy learning by doing, and I believe in sharing my progress with the world.

Why I Created This Website:

I wanted a single, well-structured platform where all my learning projects could live together. Creating this site not only helped me practice real-life website structure and responsiveness, but also gave me a chance to build something I can continuously improve as I grow in my coding journey.
